Astro é o framework web para a construção de websites focados em conteúdo. Futuramente irei fazer um novo post sobre a ferramenta e trazer mais detalhes sobre suas particularidades.

## Cabeçalhos
Utilizamos o # para indicar um cabeçalho

# h1 cabeçalho

## h2 cabeçalho

### h3 cabeçalho

#### h4 cabeçalho

##### h5 cabeçalho

###### h6 cabeçalho

## Linhas Horizontais
Utilizamos --- (três traços) para indicar linhas como essa:

---


## Ênfase

Utilizamos ** (dois asteriscos) antes e depois do texto que queremos deixar em negrito.

- **Esse é um texto em negrito**

Utilizamos _ (underline) antes e depois do texto que queremos deixar em itálico.

- _Esse é um texto em itálico_

Utilizamos ~~ (dois til) antes e depois do texto que queremos 

- ~~Esse é um texto tachado~~

## Listas

Não ordenada

- Crie uma lista iniciando uma linha com `+`, `-` ou `*`
- As sublistas são feitas inserindo 2 espaços:
  - A mudança de caractere do marcador força o início de uma nova lista:
    - Exemplo 1
    * Exemplo 2
    - Exemplo 3
- **Muito fácil**!

Ordenada

1. Apenas coloque o 1. e escreva o conteúdo.
2. Exemplo 2.
3. Exemplo 3.


## Código

Bloco de código utilizamos ``` (três acentos grave) antes e depois do código que queremos destacar:

```
Esse é um texto simples...
```

Destaque de sintaxe após os três primeiros acentos insira a sigla da linguagem que o código foi escrito:

```js
var resultado = soma(a, b) {
  return a+b;
};

console.log(soma(5, 5));
```

Atualmente apenas a linguagem JavaScript tem o suporte para sintaxe destacada.
